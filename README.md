# SampleFuel
Network requests with Fuel in Android

## Requirements

* Android Studio 3.3 Canary 13
* Kotlin 1.3.0-rc-116
* Android Gradle Plugin 3.3.0-alpha13
* Gradle wrapper 4.10.1
* AAPT 2
